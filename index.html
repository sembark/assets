Favicons

Favicons are special cases of icons that are using as icons for browsers tabs. We have SVGs and PNGs icons for light and dark theme (system preference).

Source Description Preview
favicons/favicon.svg SVG favicon for light theme
favicons/favicon.png PNG favicon for light theme
favicons/favicon-dark.svg SVG favicon for dark theme
favicons/favicon-dark.png PNG favicon for dark theme

To add support for light/dark theme icons, we need to add the PNG and SVG favicons to the <head> tag. 


    <!-- Fallback icon if SVG icons are not supported -->>
    <link
      rel="alternate icon"
      class="js-site-favicon"
      type="image/png"
      href="favicons/favicon.png"
    />
    <!-- SVG icon -->
    <link
      rel="icon"
      class="js-site-favicon"
      type="image/svg+xml"
      href="favicons/favicon.svg"
    />

We use JavaScript to update the favicons based on current theme along with listening to the theme changes. The script is available at js/themed-favicon.js. You can link the provided JavaScript file to handle the themed favicon. 


    <script src="js/themed-favicon.js"></script>

Finally, we have fallback .ico file for favicon at favicons/favicon.ico
